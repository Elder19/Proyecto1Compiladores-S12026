    ¡¡ error lexico: '?' es invalido
    int ~ x <- 5?!

    ¡¡ error lexico: '`' es invalido  
    float ~ y <- 3.14`!

    ¡¡ error sintactico: falta '!' al final
    int ~ z <- 10

    ¡¡ esto se recupera y sigue
    bool ~ activo <- true!

    ¡¡ error sintactico: asignacion sin valor
    int ~ w <- !
